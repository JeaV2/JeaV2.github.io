Op deze pagina ga ik uitleggen welke JavaScript en CSS technieken ik heb gebruikt voor het maken van deze pagina's.

Pagina layout

De layout van de pagina's is gemaakt met CSS. De pagina's zijn opgedeeld in een container, nav en main. De container is de gehele pagina, de nav is de navigatie aan de linkerkant en de main is de content in het midden van de pagina.
De container is een grid met 3 kolommen en 2 rijen. De kolommen zijn 1fr, 3fr en 1fr breed en de rijen zijn 1fr en 9fr hoog, zodat je een navigatie aan de linkerkant hebt en de content in het midden.
Ook omdat de kolommen 1fr, 3fr en 1fr zijn, is de content in het midden 3 keer zo breed als de navigatie aan de linkerkant.
De grid-template-areas zijn "nav title ." en "nav main .". Dit zorgt ervoor dat de navigatie aan de linkerkant staat en de content in het midden.
Er zit een ruimte van 1rem tussen de grid-items en de container is 95vh hoog en 3vh vanaf de bovenkant van de pagina. 
                    
CSS 

                    
                        .container {
                            display: grid;
                            grid-template-columns: 1fr 3fr 1fr;
                            grid-template-rows: 1fr 9fr;
                            grid-template-areas: "nav title ." "nav main .";
                            grid-gap: 1rem;
                            height: 95vh;
                            margin-top: 3vh;
                        }
Dit is de CSS voor de titel, navigatie en content van de pagina.
De titel heeft een grid-area van title, de navigatie heeft een grid-area van nav en de content heeft een grid-area van main, dit zorgt ervoor dat de items op de juiste plek staan in de container. 
                    
CSS 

                    
                        #title, .nav, main {
                            background-color: var(--bg-color);
                            border: solid var(--secondary-color) 5px;
                            border-radius: 10px;
                            overflow: scroll;
                            -ms-overflow-style: none;
                            scrollbar-width: none;
                            box-shadow: var(--main-color) var(--box-b-s);
                        }

                        #title {
                            grid-area: title;
                            overflow: hidden;
                        }

                        .nav {
                            grid-area: nav;
                        }
                        .nav::-webkit-scrollbar {
                            display: none;
                        }

                        main {
                            grid-area: main;
                        }

Javascript dag deel

Aan de bovenkant van sommige pagina's staat een groet met het dagdeel erin. Dit is gedaan met JavaScript. Dit wordt gedaan door met JavaScript de huidige tijd op te vragen en te kijken welk dagdeel het is met verschillende if-statements. 
                    
JavaScript 

                    
                        const date = new Date();
                        const hours = date.getHours();
                        const dagdeel = document.getElementById('dagdeel');
                        console.log('Time:', hours);

                        if (hours >= 6 && hours < 12) {
                            dagdeel.innerHTML = 'Goedemorgen';
                        }
                        else if (hours >= 12 && hours < 18) {
                            dagdeel.innerHTML = 'Goedemiddag';
                        }
                        else if (hours >= 18 && hours < 24) {
                            dagdeel.innerHTML = 'Goedenavond';
                        }
                        else { dagdeel.innerHTML = 'Goedenacht'; }

Random quote

Op de homepagina staat een random quote. Deze quote wordt opgehaald uit een array met quotes en wordt random gekozen met JavaScript.
Er wordt een random getal gegenereerd met Math.random() en Math.floor() en de lengte van de array, met dat getal wordt een quote uit de array gehaald, waarna het in de HTML wordt gezet met innerHTML. 
                    
JavaScript 

                    
                        const quotes = {
                            "quotes": [
                                {
                                    "quote": "Coding like poetry should be short and concise.",
                                    "author": "Santosh Kalwar"
                                },
                                ...
                                {
                                    "quote": "Computers are fast; developers keep them slow.",
                                    "author": "Anonymous"
                                }
                            ]
                        }
                        
                        const quote = document.getElementById('quote');
                        
                        function randomQuote() {
                            const i = Math.floor(Math.random() * quotes.quotes.length);
                            quote.innerHTML = `"${quotes.quotes[i].quote} <sup>${quotes.quotes[i].author}</sup>"`;
                        }
                        
                        randomQuote();

Codeblock buttons

De knopjes op de code blocks verstoppen en laten zien van de code blocks is gedaan met JavaScript.
Eert wordt er een querySelectorAll gebruikt om alle code blocks te selecteren. Daarna wordt net een functie het juiste code block getoggled tussen display: none en display: block.
Met een if-statement wordt gekeken of de display van het code block none is, als dat zo is wordt het block block en anders none. 
                    
JavaScript 

                    
                        const codeblock = document.querySelectorAll('code');
                        console.log(codeblock);
    
                        function toggleCodeblock(i) {
                            if (codeblock[i].style.display === 'none') {
                                codeblock[i].style.display = 'block';
                            } else {
                                codeblock[i].style.display = 'none';
                            }
                        }
