Feedback en actieplan

De feedback die ik over mijn website heb gekregen is dat het er mooi en leuk uitziet. Ook vonden studenten de codeblock waar ik de code in uitleg een leuke toevoeging.
Daarnaast had ik de feedback gekregen om het CRT-effect minder te maken omdat mensen daar last van zouden kunnen hebben, en dat ik de linkerkant moest opvullen omdat deze leeg was.

Ik heb de verbeterpunten aangepakt, ik heb het CRT-effect minder gemaakt en de linkerkant opgevuld met bewegende wireframes.

Wat ik de volgende keer beter kan doen is eerder aan mensen vragen of ze last hebben van de CSS-effecten op de website, zodat ik dit eerder kan aanpassen.

Persoonlijke leerdoelen

De dingen die ik in de volgende periode wil leren zijn:
  • Ik wil graag leren hoe ik een domeinnaam kan kopen.
  • Ik wil graag leren hoe de backend van websites werken.
  • Ik wil graag nog meer programmeertalen leren te gebruiken.
  • Ik wil graag leren hoe je een server aan een domeinnaam kan koppelen.
