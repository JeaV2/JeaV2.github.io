Moeilijkheden en uitdagingen.

Ik heb eigenlijk geen grote moeilijkheden of uitdagingen gehad met het maken van deze website. Ik heb wel een paar kleine dingen gehad die ik niet helemaal snapte, maar dat was snel opgelost.
Waar ik tegen aan liep was een # vergeten in een CSS bestand, waardoor bepaalde elementen niet goed werden weergegeven.
Dit heb ik na enige tijd gevonden en opgelost.
Ook had ik een probleem met de style voor de <pre> en <code> elementen. De code hield namelijk de indentatie van het HTML bestand aan.

Hulpmiddelen

Om het probleem met de <pre> en <code> elementen op te lossen heb ik online gekeken naar hoe ik dit kon oplossen.
Hierdoor kwam ik op StackOverflow uit en heb ik daar een oplossing gevonden. 
                    
CSS 🗕🗙

                    
                        $("pre code").each(function(){
                            var html = $(this).html();
                            var pattern = html.match(/\s*\n[\t\s]*/);
                            $(this).html(html.replace(new RegExp(pattern, "g"),'\n'));
                        });
Dit stukje JavaScript zorgt ervoor dat de indentatie van de code wordt verwijderd en de juiste indentatie wordt toegepast.
Dit stukje is in jQuery geschreven, ik snap daarom maar gedeelelijk hoe dit werkt.
