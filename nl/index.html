Wie ben ik?

Goedendag, Ik ben Jea, een student Creative Software Developer aan het Grafisch Lyceum Rotterdam. Ik zit momenteel in mijn tweede jaar!
Voordat ik begon aan het Grafisch Lyceum, was ik al geïnteresseerd in programmeren en web ontwikkeling. Ik volgde een cursus over Python op de middelbare school en ik was al bezig met het leren van HTML, JavaScript en CSS in mijn vrije tijd.
Op school heb ik mijn skills in HTML, CSS en JavaScript verder ontwikkeld. Ook ben heb ik kennis gemaakt met PHP en databases.
Naast programmeren en webontwikkeling hou ik ook van gamen, pixel art tekenen, lezen en slapen!
Mijn favoriete games op dit moment zijn Minecraft, Noita en NIGHT-RUNNERS PROLOGUE.
In mijn vrije tijd werk ik aan kleine persoonlijke projecten, zoals websites en pixel art. Wanneer ik de kans krijg, skate ik ook af en toe.

Wat kan ik?

Ik ben al twee jaar bezig met het leren van HTML, CSS en JavaScript op het Grafisch Lyceum en iets langer in mijn eigen tijd. Ik weet hoe ik een website vanaf niks moet maken, en ik ben bekend met DOM-manipulatie, fetch API en asynchrone programmering.
Ik ben ook ongeveer anderhalf jaar bezig met het leren van PHP, en ik weet hoe ik een eenvoudige backend ermee kan maken. Ik heb enkele kleine schoolprojecten gemaakt met PHP en Sqlite- en PhpMyAdmin-databases.
Ik weet ook hoe ik JavaScript libraries zoals P5.js en Three.js kan gebruiken om interactieve graphics en animaties te maken.
Ik heb ook ervaring met CSS frameworks zoals Bootstrap en ik weet hoe ik Git en GitHub kan gebruiken voor versiebeheer.
(Deze website is gemaakt met bootstrap, HTML, wat custom CSS en wat JavaScript)
Ik heb ook wat ervaring met Python. Ik heb enkele kleine projecten gemaakt met Python, ik heb het vooral gebruikt voor kleine persoonlijke projecten en één schoolproject. personal projects and one school project.
Ik ben altijd enthousiast om nieuwe dingen te leren en mijn vaardigheden te verbeteren. Momenteel leer ik meer over PHP, databases en EcmaScript 6.

Waar ik wil zijn

In de toekomst zou ik graag ergens als webontwikkelaar of softwareontwikkelaar willen werken. Ik werk het liefst aan de backend en met echte programmeertalen. Ik vind het echter ook leuk om aan de frontend te werken en dingen er af en toe mooi uit te laten zien (zoals je op deze website kunt zien).

Wat ik zou willen leren

Tijdens een stage of een baan zou ik graag meer willen leren over PHP en PHP-frameworks zoals Laravel aangezien ik nog niet de kans heb gehad om ermee te werken.
Ik zou ook graag willen leren hoe echte productiecodebases eruit zien en werken en hoe je in een team van developers werkt, aangezien we dat op school hebben nagebootst, maar het is niet hetzelfde als de echte situatie.
