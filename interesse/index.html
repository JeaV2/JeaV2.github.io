Interesses

Ik zou zelf graag meer willen leren over de volgende onderwerpen:
  • Hoe ik een eigen domain name kan kopen.
  • Hoe ik een VPS (Virtual Private Server) kan kopen om een website op te hosten.
  • Leren hoe ik een backend kan hosten op die VPS.
  • Meer leren over andere programmeertalen en waar ik deze voor kan gebruiken.

Redenen

Ik wil graag meer leren over deze onderwerpen, omdat ik het interessant vind om te weten hoe ik een website kan hosten en hoe ik een backend kan maken.
Ook lijkt het me leuk om te weten hoe ik een eigen domain name kan kopen en hoe ik deze kan koppelen aan een website, omdat ik nu nog gebruik maak van een subdomain van GitHub.
Een eigen domain name ziet er professioneler uit en is makkelijker te onthouden.
Ook lijkt het me leuk om meer te leren over andere programmeertalen, omdat ik nu vooral gebruik maak van HTML, CSS en JavaScript, maar er zijn nog veel meer programmeertalen die ik kan leren.

Eigen leerpad

Om meer te leren over deze onderwerpen wil ik graag een eigen leerpad maken.
Dit leerpad bestaat uit de volgende stappen:
  1. Uitzoeken hoe ik een domain name kan kopen.
  2. Uitzoeken hoe ik een VPS kan kopen.
  3. Uitzoeken hoe ik een backend kan hosten op die VPS.
  4. Uitzoeken welke programmeertalen ik wil leren en waar ik deze voor kan gebruiken.

Bronnen

Ik heb al een keer een website gevonden over hoe je een domain name kan kopen, en een server kan koppelen. Maar ik heb er nog niks mee gedaan.
Ook is de tutorial meer afgesteld op Amerikaanse mensen als ik het mij goed herinner.
